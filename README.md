*群眾募資*
* [創夢市集](https://www.ditfunding.com) - 是由五家遊戲公司樂陞、遊戲橘子、華義、昱泉、網銀國際共同出資的新創公司,希望集結台灣遊戲、行動應用、文化創意產業的創業能量，透過「創業育成」、「群眾募資」與「創業投資」等三大服務，提供全方位的發展規劃，支持新創產業
